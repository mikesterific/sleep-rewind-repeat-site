# Sleep Rewind Repeat Site

Small public marketing, support, and privacy site for Sleep Rewind Repeat.

## Pages

- Home: `index.html`
- Support: `support.html`
- Privacy Policy: `privacy-policy.html`

## Suggested GitHub Pages Setup

Create a new public GitHub repository named:

`sleep-rewind-repeat-site`

Then from this folder:

```bash
git init
git add .
git commit -m "Initial Sleep Rewind Repeat site"
git branch -M main
git remote add origin https://github.com/mikesterific/sleep-rewind-repeat-site.git
git push -u origin main
```

In GitHub:

1. Open the repository settings.
2. Go to Pages.
3. Set Source to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save.

Expected URLs:

- Marketing site: `https://mikesterific.github.io/sleep-rewind-repeat-site/`
- Support URL: `https://mikesterific.github.io/sleep-rewind-repeat-site/support.html`
- Privacy Policy URL: `https://mikesterific.github.io/sleep-rewind-repeat-site/privacy-policy.html`

Use the support and privacy URLs in App Store Connect.
